# Fome Zero

Projeto web para conectar pessoas e organizações na doação de alimentos, ajudando a reduzir o desperdício e combater a fome.

---

## Sobre o projeto

O **Fome Zero** é uma aplicação front-end que permite:

* Publicar doações de alimentos
* Conectar doadores com pessoas/ONGs
* Incentivar o reaproveitamento de alimentos

---

## Tecnologias utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

---

## Páginas do projeto

* **Home (`index.html`)**

  * Apresentação do projeto
  * Explicação de como funciona
  * Estatísticas de impacto

* **Doar alimento (`registerFood.html`)**

  * Formulário para cadastro de doações
  * Feedback com toast de sucesso
